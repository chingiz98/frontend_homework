
## Frontend для модульстарт
### Реализованные функции:
- Пополнение счета
- Переводы между своими счетами и чужими
- История операций с фильтрами по дате
- Открытие счетов
- Закрытие счетов
- Регистарция
- Авторизация
- Изменение данных пользователя
